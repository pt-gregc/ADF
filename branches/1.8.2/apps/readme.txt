    ADF APPLICATIONS


Custom built applications that can be a combination of components (cfc's), 
custom coding (custom field types, custom scripts, renderhandlers etc...) 
and common ADF library components.

The applications will be placed in this "/ADF/apps" directory.

Documentation for ADF Applications are located in the ADF Docs wiki site 
(http://community.paperthin.com/adf-docs/index.php/ADF_Applications).

Applications to download and install are located in the PaperThin Community Site 
("The Commons") (http://community.paperthin.com/projects/).
